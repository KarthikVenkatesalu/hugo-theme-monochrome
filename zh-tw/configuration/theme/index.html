主題設定

這些設定是在網站的 toml 設定檔裡面

Navbar title

支援 HTML

1
2

[params]
navbarTitle = "Graytr"

default: site’s title

Footer

支援 HTML

1
2

[params]
footer = "Copyright © 2021 by kaiiiz"

default: footer 不會被渲染

Table of Contents

1
2

[params]
enableTOC = false

default: true

Navbar menu

Graytr 使用 hugo 內建的 menu 系統，參考 官方文件。預設 navbar 這個 menu 會被拿來渲染到頂部。

最簡單的 menu 使用模式是透過將以下設定加到某個頁面中的 front-matter，hugo 即會將該頁面加入 navbar 這個 menu 中。(simple method mentioned in document)

1
2
3

---
menu: navbar
---

預設 hugo 支援權重，可以明確指定權重來調整在 navbar 中的順序，如果有兩個權重相同的頁面，將會使用字典序排序。

1
2
3
4
5

---
menu:
  navbar:
    weight: -99
---

更進階的 menu 應用請參考 Add Non-content Entries to a Menu，你也可以參考這個網站的目錄設定。

Syntax Highlighting

1
2

default: true

Prism.js (client-side)

關閉 markup highlight

1
2
3

[markup.highlight]
codeFences = false
lineNos = false

1
2

[params]
usePrismJS = true

default: false

啟用行數功能

1
2

[params]
enablePrismJSLineNo = true

default: true
