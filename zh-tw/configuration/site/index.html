網站設定

參考 All Configuration Settings

Site title

網站標題

1

title = "Graytr"

Theme

設定主題的名稱（在 /themes 下的目錄名稱）

1

theme = "hugo-theme-graytr"

Base URL

Hostname 與 path

1

baseURL = "https://kaiiiz.github.io/hugo-theme-graytr"

favicon

將 favicon.ico 加到網站的 static 目錄.

disqusShortname = "xxx"
