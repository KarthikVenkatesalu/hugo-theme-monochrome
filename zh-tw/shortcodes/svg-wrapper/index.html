SVG Wrapper

將 SVG 包裝上一層顏色的樣式，可以提升對黑色主題的兼容性。

使用

{{< svg-wrapper name="" link="" >}}
svg inline
{{< /svg-wrapper >}}

參數:

  • link: 渲染連結
  • name: 使用主題內建的 SVG
